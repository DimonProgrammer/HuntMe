### 1.1 Instagram — Comment-to-DM (безопасный путь)

**Как работает:** пост с CTA → пользователь комментирует ключевое слово → авто-DM за <3 секунды.

| Параметр | Значение |
|---|---|
| Инструмент | **ManyChat** ($15/мес) или **Inro** (free plan) |
| Тип трафика | Только те, кто сам проявил интерес |
| DM open rate | **80-100%** (vs 20% email) |
| Уровень риска бана | **Низкий** — официальный Meta API |
| Суточный лимит | Нет фиксированного лимита для входящих |
| Setup time | 1-2 часа |

**Сценарий для BR моделей:**
- Пост в IG Reels: "Trabalha de casa em dolares? Comenta QUERO e te explico tudo 🇧🇷"
- ManyChat: comment trigger "QUERO" → авто-DM с коротким описанием + ссылка на TG-бот

**Риски + обходы:**
- ❗ Холодные DM рандомным людям (без их инициативы) — официальный API это не позволяет
- ❗ Unofficial tools (AutoReacher, Jarvee) — утверждают 120K DM/мес, аккаунт клонирование → **перманентный бан**
- ✅ Только comment-to-DM или reply-to-Story как триггеры
- ✅ Разнообразить тексты постов (не копипастить)
- ✅ Business/Creator аккаунт обязателен

**Воронка:**
```
IG Reel → Комментарий "QUERO" → Auto-DM → Ссылка на TG-бот / WhatsApp
```

### 1.2 Facebook Groups — Органический постинг + мониторинг

**Два безопасных подхода:**

**Подход A — Создать свою группу + GroupBoss:**
- Создать FB-группу (например, "Renda Extra em Dolares — Brasil")
- В вопросах при вступлении спросить: email, cidade, tem camera?
- **GroupBoss** автоматически захватывает ответы → Google Sheets или CRM
- Результат: квалифицированные лиды без скрейпинга

**Подход B — Мониторинг чужих групп через Devi AI:**
- Вступить в 15+ групп (trabalho remoto, renda extra, mulheres empreendedoras BR)
- **Devi AI** отслеживает посты с ключевыми словами ("preciso de renda", "trabalho em casa")
- Получаешь alert → отвечаешь контекстно, не холодно
- Конверсия на 30% выше чем холодный аутрич

| Параметр | GroupBoss | Devi AI |
|---|---|---|
| Цена | ~$19/мес | ~$19/мес |
| Уровень риска | **Минимальный** | Низкий |
| Скорость лидов | Пассивно | Реактивно |
| Лидов/мес | 20-50 (зависит от роста группы) | 10-30 intent-based |

**❌ Что НЕ делать:**
- Скрейпить участников групп через Apify без прокси → FB временно блокирует
- Массово добавлять людей в личку без инициативы с их стороны
- Спамить один пост во 100 групп одновременно (спам-детекция)

**✅ Безопасный ритм постинга:**
- 1 группа/день прогрева (лайки, комменты)
- После 3-5 дней → 1 пост/неделю на группу
- Ротация из 5-7 разных вариантов текста

## КЛЮЧЕВЫЕ ВЫВОДЫ (HuntMe модель)

1. **HuntMe $10/день = recurring.** Это не one-time PPS как Chaturbate $50. Каждый рабочий день модели = $10. За 1 месяц (20 дней) = $200-275 с учётом бонусов. В 4-5x больше чем прямые аффилиатки.

2. **Даже пессимистичный сценарий прибыльный.** Break-even CPL $4.40 > Brazil Ads CPL $1.50-3. Запас прочности есть.

3. **Retention = главный рычаг.** Каждая retained модель = $160/мес пассива (80% от $200). 10 retained = $1,600/мес. 20 = $3,200/мес. Кумулятивный эффект огромный.

4. **Organic-first остаётся оптимальным стартом.** Не потому что paid убыточен, а потому что нужно замерить реальную конверсию шок-раскрытия (40-60% уходят).

5. **Click-to-WhatsApp BR = лучший paid канал.** CPL $1-2.50, на 50% дешевле обычных конверсий. При подтверждённой конверсии — масштабировать агрессивно.

6. **При $1K/мес рекламы (neutral): $7,640 net мне.** К месяцу 12 с retention tail: **$23K/мес.** Это серьёзный бизнес.

7. **Реферальная программа ($20 бонус)** — при HuntMe модели экономика ещё лучше: платишь $20, получаешь $200-275 за месяц работы реферала. ROI 1,000%+.

8. **Главный риск: конверсия < 2%.** Если из 100 лидов только 1 модель реально начинает работать — marginal на paid. Поэтому organic test first.

9. **Не нужно разбираться с платформами.** Студии-партнёры HuntMe сами решают куда сажать модель, сами помогают с ID-верификацией, настройкой оборудования. Наша задача = только лидген + скрининг + довести до собеседования.
