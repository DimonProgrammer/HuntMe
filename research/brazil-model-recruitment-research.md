## Executive Summary

1. **Бразилия — крупнейший нетронутый рынок webcam-моделей**: 212M население, 10-е место в мире по потреблению adult-контента, 22M+ пользователей, при этом мало студий (в отличие от Колумбии с 12K+).
2. **Оффер для модели: $500-2,000/мес на платформе** (R$2,900-11,600) — сопоставим или кратно выше средней зарплаты женщин (R$2,640/мес по IBGE). Наш доход: $10/день за каждый рабочий день модели = $150-200/мес с одной модели.
3. **WhatsApp — #1 канал** (93.7% проникновение), Click-to-WhatsApp реклама через Meta Ads — оптимальная воронка при CPL ~$1.50-3.00.
4. **Главный барьер — шок-фактор** (раскрытие что это camgirl): ожидаемый dropout 40-60%. Двухступенчатое раскрытие критично.
5. **Вердикт: GO (Pilot)** — при бюджете $100/нед достаточно 3 модели/мес для break-even. Реалистично даже в пессимистичном сценарии.

## 4. Оффер (Hormozi Value Equation)

```
Value = (Dream Outcome × Perceived Likelihood) / (Time Delay × Effort & Sacrifice)
```

### Заполнение формулы

| Компонент | Наш оффер | Оценка |
|-----------|-----------|--------|
| **Dream Outcome** | Заработок $500-2,000/мес в USD, работая из дома, гибкий график, анонимность (geo-block BR) | **9/10** |
| **Perceived Likelihood** | Есть оператор-наставник, поддержка 24/7, обучение с нуля, платформы с многомиллионной аудиторией | **6/10** (нужен social proof) |
| **Time Delay** | Первый заработок через 3-7 дней после старта. Обучение 1-2 дня. Регистрация 30 минут | **8/10** |
| **Effort & Sacrifice** | Нужен только телефон/ноутбук + интернет + камера. Стигма — главная "жертва" | **5/10** (стигма тянет вниз) |

**Value Score: (9 × 6) / (2 × 5) = 54 / 10 = 5.4** — хороший, но стигма снижает. Нужно работать над Perceived Likelihood (social proof) и снижать Effort (антистигма-messaging).

### Формулировка оффера

**PT-BR (нейтральный):**

## Решение: GO (Pilot)

### Аргументация

**ЗА (Go):**
- Рынок огромный (212M), цифрово зрелый (87% интернет, 93.7% WA)
- Webcam-рекрутинг в BR практически пустой (vs Колумбия с 12K+ студий)
- Наш оффер в USD = 2× средней женской зарплаты при работе из дома
- CPL в Meta Ads = $1.50-3.00 → break-even при 3 моделях/мес
- Легальный фреймворк (CNAE 96090), нет юридических рисков
- Click-to-WhatsApp → двухступенчатое раскрытие — уникальная стратегия

**ПРОТИВ (риски):**
- Шок-фактор: 40-60% dropout после раскрытия формата. Нужно много лидов
- Язык: нужен контент на PT-BR (не испанский). Нужна помощь носителя или AI-перевод
- Meta модерация: ads с даже косвенным adult-подтекстом могут банить. Нужен нейтральный креатив
- Нет social proof для BR-рынка (пока). Первые 5-10 моделей = самые дорогие

**Рекомендация:**
1. **Неделя 1-2:** Органический тест ($0-20). IG аккаунт + 5 Reels + FB Groups + WA Groups
2. **Неделя 3-4:** Платный тест ($100/нед). Meta Ads Click-to-WhatsApp. Замерить CPL и конверсию
3. **Месяц 2+:** Масштабирование при CPL < $3 и конверсия > 3%

**Break-even анализ:**
- Расходы: $430/мес (ads) + $135 (tools) = $565
- Доход с 1 модели: ~$200/мес (нам 80% = $160)
- Break-even: **4 модели/мес** ($565 ÷ $160 = 3.5) = ~3% конверсия из ~140 лидов → **реалистично**
