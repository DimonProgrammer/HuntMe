### 1.3 Юнит-экономика на $100/неделю ($430/мес)

**Доход с 1 модели (работает 1 месяц):**
- 15-20 рабочих дней × $10/день = $150-200 gross
- Бонус за 2-ю смену: $50 (LatAm)
- Итого: $200-275 gross → мне (80%): **$160-220**

**При $100/нед на рекламу (Meta Click-to-WhatsApp):**

| Метрика | Пессимист | Нейтрал |
|---------|-----------|---------|
| Лидов/мес (CPL $2-3) | 140-215 | 140-215 |
| Моделей активных | 4-6 | 11-17 |
| Gross revenue | $800-1,650 | $3,025-4,675 |
| Мне (80%) | $640-1,320 | $2,420-3,740 |
| Минус ads ($430) + tools (~$135) | -$565 | -$565 |
| **NET мне** | **$75-755** | **$1,855-3,175** |
| **ROI** | **13-134%** | **328-562%** |

**Break-even:** нужно **4 модели/мес** при расходах $565 ($565 ÷ $160 = 3.5). Это ~2-3% конверсия из лидов — реалистично даже в пессимистичном сценарии.

### 2.1 Активация model_flow (Phase 2)

**Текущее состояние:**
- `bot/handlers/model_flow.py` — **готов** (7 шагов FSM: name → age → region → english → platform_experience → schedule → contact)
- `bot/handlers/agent_flow.py` — **готов** (6 шагов: name → region → english → experience → hours → contact)
- Оба НЕ подключены в `bot/main.py`
- Меню не даёт выбор роли — сразу идёт в operator_flow
- Admin callbacks для model/agent НЕ реализованы

**Что нужно сделать:**

#### A. Подключить роутеры в main.py

Файл: `bot/main.py`
- Добавить импорты: `from bot.handlers import model_flow, agent_flow`
- Добавить роутеры: `dp.include_router(agent_flow.router)` и `dp.include_router(model_flow.router)` перед operator_flow

#### B. Меню выбора роли в menu.py

Файл: `bot/handlers/menu.py`
- После `/start` → показать кнопки выбора: "Operator" / "Model" (/ "Agent" — позже)
- По выбору → создать Candidate с правильным `candidate_type` → запустить соответствующий flow
- Для deep links с лендинга: определять тип по префиксу (land_model_*, land_agent_*)

#### C. Admin callbacks для model/agent

Файл: `bot/handlers/admin.py`
- Добавить обработчики: `modelok_{user_id}`, `modelno_{user_id}`, `agentok_{user_id}`, `agentno_{user_id}`
- Аналогичная логика как для операторов

#### D. Сообщения для model flow

Файл: `bot/messages/en.py` (+ `ru.py`)
- Добавить model-specific greeting, step prompts, decline messages
- Опционально: `pt.py` для португальского (Phase 2)
