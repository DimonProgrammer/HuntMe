## CATEGORY A: Model Candidate

### Criteria (check all that apply)

- [ ] **Female** (mandatory)
- [ ] **Age 18-35** (check profile or cover letter — if age not stated, infer from photo/profile)
- [ ] **Has profile photo** showing appearance (clear face photo, photogenic/attractive)
- [ ] **Has smartphone or webcam** (mentioned in profile or can be assumed)
- [ ] **Stable internet** (stated or implied by location — urban PH/LatAm = likely yes)
- [ ] **Open to creative/content/streaming work** (mentions: content creation, social media, streaming, entertainment, hosting, modeling, influencer, vlogging, TikTok)
- [ ] **Communicative profile** (cover letter shows personality, warmth, energy)
- [ ] **Based in PH, LatAm, or other target region**

### Green Flags (prioritize these)
- Mentions experience with TikTok, Instagram, YouTube, or any streaming
- Has a social media presence with followers
- Cover letter shows personality and charisma
- Mentions "work from home," "flexible schedule," or "remote"
- Has experience in customer service, hospitality, or entertainment
- Profile photo is high quality, confident, well-presented

### Red Flags (deprioritize or skip)
- No photo at all and no social media links
- Profile text is copy-pasted or clearly mass-applied
- Under 18
- Male (redirect to Category B or C)
- Mentions explicit adult content interest
- Location in restricted country

### Action
Send **Template A** response (below), directing to Telegram bot with model deep link.

## CATEGORY B: Agent Candidate

### Criteria (check 3+ of these)

- [ ] **Any gender** (male or female)
- [ ] **Has social media presence** or mentions managing social accounts
- [ ] **Recruitment experience** (mentions: recruiting, talent acquisition, HR, headhunting, staffing)
- [ ] **Sales or outreach experience** (mentions: lead generation, sales, outreach, cold calling, telemarketing)
- [ ] **Good communication skills** (cover letter is well-written, persuasive, detailed)
- [ ] **Entrepreneurial mindset** (mentions: passive income, freelancing, side hustle, business, self-starter)
- [ ] **Community management** (mentions: Facebook groups, WhatsApp groups, community, moderator)
- [ ] **VA experience** with client communication skills

### Green Flags
- Large social media following (any platform)
- Experience managing Facebook/WhatsApp groups
- Cover letter mentions wanting "passive income" or "multiple income streams"
- Experience in MLM/network marketing (they already know how to recruit, just needs a legitimate offer)
- Mentions being a "people person" or "connector"
- Has a personal brand or blog

### Red Flags
- Zero social media presence AND no communication skills evident
- Expects fixed salary (this is performance-based)
- Cover letter is one sentence with no effort

Send **Template B** response (below), directing to Telegram bot with agent deep link.

## CATEGORY C: Operator Candidate

### Criteria (ALL of these must be true)

- [ ] **Male** (mandatory — females should be redirected to Category A)
- [ ] **Age 18-30** (stated or implied)
- [ ] **Technical skills** (mentions: PC, computer, IT, tech support, gaming, streaming software, OBS)
- [ ] **Has PC with discrete GPU** (mentions: desktop, gaming PC, NVIDIA, AMD, graphics card — or at minimum has Windows PC)
- [ ] **English B1+** (cover letter is coherent and grammatically reasonable)
- [ ] **Available to start within 1 week** (or flexible availability)

### Green Flags (prioritize)
- Mentions OBS Studio, Streamlabs, or streaming software
- Gaming background (familiar with Twitch, Discord, streaming culture)
- Experience in chat moderation, customer support, or live events
- Has Windows PC with specs mentioned (i3 10th+, GTX 1060+, 100Mbps+)
- Cover letter shows attention to detail and structured thinking
- Mentions 5/2 schedule availability

### Red Flags
- Uses MacBook only (operators need Windows)
- No PC at all (phone-only applicant)
- Female (redirect to Category A)
- In-person student with rigid schedule
- Poor English — cannot construct basic sentences
- Location with unreliable internet infrastructure

### Action
Send **Template C** response (below), directing to Telegram bot with operator deep link.

## CATEGORY D: Skip

### Criteria (any of these = skip)

- [ ] One-line application with no real information
- [ ] Clearly mass-applied (generic cover letter, no mention of the role)
- [ ] Profile is incomplete — no name, no photo, no skills, no experience
- [ ] Under 18 (stated or clearly apparent)
- [ ] Spam or suspicious profile
- [ ] Doesn't match any of the above categories
- [ ] Demands unrealistic salary for entry-level
- [ ] Located in restricted/unsupported region
- [ ] Profile suggests bot or fake account

### Action
No response needed. Move to next applicant.

## SPEED SORTING TIPS

1. **Start with gender.** Female = likely Category A. Male = likely Category B or C.

2. **Check the profile photo.** Female with clear photo + age 18-35 = fast-track to Category A. No photo = check cover letter for agent/operator fit.

3. **Scan the cover letter for keywords:**
   - "streaming," "content," "social media," "TikTok," "model" = Category A
   - "recruiting," "passive income," "outreach," "sales," "network" = Category B
   - "OBS," "gaming," "PC," "tech," "IT," "moderator" = Category C
   - One sentence, no effort, spam = Category D

4. **When in doubt between A and B for females:** If she has a clear photo and mentions anything content/entertainment-related, go A. If she's more business/recruiting-oriented, go B. She can be both — send Template A (model pays more for us).

5. **Batch process:** Sort all applicants into categories first (A/B/C/D). Then batch-send templates per category. This is faster than reading + responding one by one.

6. **Use the personalized note for operators.** Copy the one-liner that matches from the table above. For models and agents, the template is already complete — just replace {{NAME}}.
