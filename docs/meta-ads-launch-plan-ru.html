📋 Пошаговый план запуска

Шаг 1 — Антидетект-браузер (5 минут)

Dolphin Anty

  1. Скачать dolphin-anty.com → зарегаться
  2. Free план = 5 профилей (достаточно)
  3. Создать профиль:
    OS: macOS / Browser: Chrome / Locale: pt-BR / Timezone: America/Sao_Paulo
  4. Подключить прокси (шаг 2) к профилю
Шаг 2 — Прокси (10 минут)

Статичный residential IP из Бразилии

  1. Зайти на Proxy-Seller → купить 1 статичный residential BR-прокси (~$3/мес)
  2. Альтернатива: Bright Data или Smartproxy (Decodo)
  3. Получить данные: IP:Port:Login:Password
  4. Вставить в Dolphin Anty → профиль → Proxy
  5. Проверить: whoer.net — должен показать Бразилию
Правило: 1 прокси = 1 аккаунт. Никогда не использовать один IP для двух FB-аккаунтов. Никогда не заходить в аккаунт из обычного браузера.

Где покупать

Маркетплейсы (безопасно — эскроу)
  • DarkStore — много продавцов, отзывы, холдирование
  • Partnerkin FB Exchange — проверенные продавцы
Проверенные шопы
  • FBStore.ru — VIP KING/ПЗРД, ручной фарм 30+ дн
  • NPPRTeam.shop — фарм, aged, верифицированные
  • CrazyFB.shop — авторег, фарм, ПЗРД, BM

Форум для проверки продавцов по отзывам: FB-Killa.pro

Красные флаги — чего избегать

  • ПЗРД за $1-2 = мусор или скам
  • Продавец только в Telegram без профиля на форумах
  • Не передают cookies (перелогин = триггер бана)
  • 2FA включена, но коды не дают
  • «Гарантия навсегда» — невозможно, это FB
  • Продавец сохраняет доступ после продажи

Прогрев после покупки (обязательно!)

  1. Зайти ТОЛЬКО через Dolphin Anty с BR-прокси
  2. День 1-2: листать ленту, лайкать, вступить в 2-3 группы
  3. День 3: создать Fan Page «Apex Talent» (рекрутинг)
  4. День 4: настроить Business Manager → добавить страницу
  5. День 5: создать рекламный аккаунт (валюта USD)
Наш козырь — белый оффер. Большинство арбитражников банятся из-за контента (гемблинг, крипта, нутра). У нас рекрутинг — полностью легальный. Аккаунт живёт дольше, при бане можно апеллировать и выигрывать.

  2. Выпустить карту (Visa/Mastercard)

Шаг 5 — Meta Pixel на лендинг (20 минут)

Установка трекинга

  1. В Ads Manager → Events Manager → Data Sources → Create Pixel
  2. Скопировать base code → добавить в <head> лендинга landing/index.html
  3. На WA-кнопку повесить событие Lead: 
    onclick="fbq('track', 'Lead', {content_name: 'whatsapp_click'})"
  4. Проверить через Meta Pixel Helper (Chrome extension)
  5. UTM-метки в рекламных ссылках: 
    ?utm_source=meta&utm_medium=paid&utm_campaign={{campaign.name}}

🛡 Безопасность аккаунта

✅ Делать
  • Заходить только через Dolphin Anty
  • Один прокси = один аккаунт
  • Прогревать 3-5 дней
  • Бюджет поднимать на +20%/день макс
  • Пополнять карту по $50-100
  • Иметь 1-2 запасных аккаунта
  • Менять 1 элемент креатива каждые 3-5 дней
❌ Не делать
  • Заходить в аккаунт из обычного браузера
  • Использовать один IP для двух аккаунтов
  • Резко поднимать бюджет ($10 → $100)
  • Апеллировать при бане (создаёт permanent flag!)
  • Редактировать живые кампании (дублируй)
  • Упоминать webcam/adult в креативах
  • Пополнять карту на полный бюджет сразу

При бане аккаунта

  1. Паузим всё. НЕ апеллируем
  2. Покупаем новый аккаунт ($15-35)
  3. Новый профиль в Dolphin Anty + новый прокси
  4. Meta Pixel — переподключаем (если был на старом BM → пересоздать)
  5. Заново прогреваем 3-5 дней
  6. Запускаем те же креативы

При белом оффере (рекрутинг) баны редки. Основная причина банов — российский IP или подозрительная платёжка, а не контент.
