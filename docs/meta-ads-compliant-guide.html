Два пути: Employment Category vs Brand-фрейм

⚡
Ключевой выбор на старте. Meta требует указывать Special Ad Category "Employment" если реклама явно говорит о найме, работе или доходе. Для LatAm-рекрутинга в формате lifestyle/brand-awareness этого можно законно избежать.
Path A · Compliance

Employment Category

Если реклама явно упоминает заработок, вакансию, найм.

  • Нет таргета по полу и возрасту
  • Нет ZIP/города (только страна)
  • Нет стандартных Lookalike
  • Интересы — доступны
  • Custom Audiences — с сертификацией
Path B · Рекомендовано

Brand / Community фрейм

Реклама как lifestyle/community, без прямого mention найма.

  • Таргет по полу: женщины 18–35
  • Таргет по городу/возрасту
  • Lookalike Audiences работают
  • Интересы — полный доступ
  • Весь арсенал Meta
✓
Для Apex Talent (Бразилия, первый запуск): используем Path B. Объявление — "entretenimento digital", lifestyle, curiosity hook. Лендинг раскрывает подробности. Никакого "вакансия", "нанимаем", "earn $X/day" в самом объявлении.
⚠
Employment Category обязателен для Европы (включая Португалию). Если когда-нибудь запускаем в EU/PT — сразу категорию. Для BR/CO/AR — не обязательно при правильном фреймировании.

Что триггерит отказ — и как этого избежать

Meta проверяет одновременно: текст объявления, визуал, лендинг по URL. Любой из трёх может стать причиной отказа.

✗ Запрещено / триггерит отказ
"Are you a beautiful woman?"
"Earn $300/day — apply now!"
"We're hiring webcam models"
"Do you want to make money online?"
Слова: webcam, adult, OnlyFans в URL
Фото в контексте спальни
Before/After с доходами в объявлении
"Last spots! Apply before Friday!"
"Struggling to earn enough?"
✓ Безопасно / проходит ревью
"Criadores de conteúdo no Brasil..."
"Descubra como funciona o entretenimento digital"
"Digital entertainment creates new career paths"
"Aprenda como criadores estão crescendo"
URL: apextalent.pro (нейтральный)
Фото в beach/city/home office контексте
Цифры дохода — только на лендинге
CTA: "Saiba mais" / "Fale conosco"
"Renda média dos criadores em destaque"

Правило Personal Attributes (главный триггер)

Meta запрещает предполагать личные характеристики читателя: пол, возраст, внешность, финансовое положение. Никогда не используй "ты" в контексте этих атрибутов.

✗ Нарушение: "Are you attractive and want to earn?" — предполагает внешность
✓ OK: "Criadores com carisma estão descobrindo novas formas de renda digital."
✗ Нарушение: "Struggling to earn enough?" — предполагает финансовые трудности
✓ OK: "A maioria das pessoas no Brasil ganha R$2.500/mês. Criadores digitais estão mudando isso."

Формулы копирайтинга, которые проходят ревью

Формула 1 — Social Proof (Самая безопасная)
[Группа людей] + [достигают результата]. + [Как это работает?]
«Criadores de entretenimento digital no Brasil estão construindo carreiras de R$12K/mês. Saiba como começa.»
Формула 2 — Curiosity / Information Gap
[Удивительный факт об индустрии] + [мягкий CTA]
«O entretenimento digital é um dos segmentos que mais cresce na América Latina. Descubra se é para você.»
Формула 3 — Third-Person Problem / Solution
[Общая проблема в 3-м лице] + [решение существует]
«A maioria das pessoas no Brasil ganha R$2.500/mês. Criadores digitais estão mudando essa equação.»
Формула 4 — Identity / Aspiration (Lifestyle)
[Аспирационная идентичность] + [как они этого достигли]
«Os criadores que estão redefinindo o entretenimento no Brasil têm uma coisa em comum: eles começaram.»
Формула 5 — Community / Belonging
«Junte-se» + [число] + [нейтральное описание группы] + [результат]
«Junte-se aos criadores de entretenimento digital que estão crescendo no Brasil este ano.»

Прогрев аккаунта и здоровье Ads Manager

🚫
Новый Business Manager + сразу агрессивный рекрутинговый таргет = быстрый бан. Meta отслеживает накопительный рейтинг качества. Нужен прогрев.
1–2

Недели 1–2: Прогрев Brand Awareness

  • 100% безопасный контент: lifestyle, digital creator culture
  • Бюджет $5–10/день, цель: Reach или Brand Awareness
  • Даём алгоритму накапливать чистую историю
  • Проверяем Account Quality dashboard ежедневно
3–4

Недели 3–4: Traffic на лендинг

  • Цель: Traffic на apextalent.pro
  • Следим за флагами в Account Quality
  • Тестируем 3–5 вариантов копи
  • Pixel собирает данные для обучения
5+

Неделя 5+: Lead Gen и конверсии

  • Click-to-WhatsApp или Lead Ad с формой
  • Бюджет наращивать не более +20%/день
  • Тест Advantage+ после 50+ лидов собрано

До первого объявления: настройка аккаунта

  1. Создать Business Manager (не личный профиль). Верифицировать бизнес.
  2. Создать Facebook Page для Apex Talent. SFW описание, логотип, контакты.
  3. Опубликовать 5–10 органических постов ПЕРЕД запуском рекламы.
  4. Привязать Instagram-аккаунт Apex Talent к BM.
  5. Установить Pixel на apextalent.pro и проверить Events Manager.
  6. Пройти Business Verification (паспорт / документы компании).

Pixel, Conversions API, события

7+/10
Event Match Quality Score — порог для эффективной оптимизации
50+
конверсий в неделю нужно для обучения алгоритма (Lead события)
-50%
CPA при правильной настройке CAPI + Pixel (источники: кейсы агентств)

Минимальный набор событий Pixel

PageView Автоматически при посещении лендинга
Lead Клик на WA-кнопку или отправка формы
Contact Начало диалога в боте (webhook обратно в Pixel)
CompleteRegistration Заявка принята, кандидат записан на интервью

Conversions API (CAPI) — обязательно

Браузерный Pixel теряет данные из-за iOS ATT и adblock. CAPI отправляет события server-side → выше EMQ → лучше оптимизация.

Для бота на Render: при успешной подаче заявки → POST в Meta CAPI endpoint с событием Lead + параметры пользователя (email/phone хеш если есть).

💡
Для Click-to-WhatsApp кампаний Meta автоматически отслеживает "Messaging Conversations Started" — CAPI для этого формата настраивать отдельно не нужно.

Форматы и креативы

Приоритет форматов (2026)

★★★ Click-to-WhatsApp — основной для BR. Открывает WA-диалог напрямую. Обходит лендинг.
★★★ Instagram / Facebook Reels (9:16) — наибольший охват, приоритизирован алгоритмом. 200B просмотров/день.
★★☆ Lead Ads (Instant Form) — форма внутри Meta, не требует лендинга. Меньше трения.
★★☆ Single Image (Feed) — дёшево, хорошо для тестирования копи.
★☆☆ Carousel — для narrative-структуры (Проблема → Процесс → Результат → CTA).

Reels: формула крео по плейбуку Meta

  1. 0–3 сек: Хук с результатом/аутпутом — не интро. "Gabriela era vendedora, hoje faz R$12K/mês com entretenimento digital."
  2. 3–20 сек: Контекст и процесс. Как она начала. Без лишних деталей о формате.
  3. 20–30 сек: Мягкий CTA. "Quer saber se é para você? Clique e conheça."
  4. Технически: 9:16, субтитры обязательно (60% смотрят без звука), реальные люди (не стоки).

Протокол при отказе объявления

  1. Читаем конкретную причину. Ads Manager → Account Quality (Business Support Home). Причина говорит, что именно флагнуто: копи, визуал, лендинг.
  2. Редактируем существующее — не создаём новое. Создание нового объявления вместо правки существующего может сигнализировать о попытке обхода. Исправить конкретный элемент и переотправить.
  3. Апелляция (если уверены в compliance). Коротко и профессионально:
    "This ad was rejected citing [policy name]. Our copy discusses creators as a general group, not individual viewer attributes. We believe this is a false positive and request manual review."
  4. Если апелляция отклонена — существенно переработать крео: другой угол, другой визуал, другая формула копи.
  5. Не подавать апелляцию повторно на тот же неизменённый креатив. Это повышает скрутинизацию аккаунта.
  6. При накопленных ограничениях аккаунта — немедленно остановить ВСЕ активные объявления. Провести аудит. Устранить нарушения, затем возобновить с консервативными крео.
🚫
Не открывать новый рекламный аккаунт при ограничениях. Meta это детектирует и эскалирует до перманентного бана Business Manager.

Чеклист перед запуском

Аккаунт и техника

  • Business Manager создан и верифицирован
  • Facebook Page настроена: описание, фото, 5+ постов
  • Instagram подключён к BM
  • Meta Pixel установлен на apextalent.pro
  • События PageView + Lead + Contact настроены
  • Conversions API подключён (или хотя бы браузерный Pixel)
  • Payment method добавлен, аккаунт активен

Лендинг

  • HTTPS без предупреждений браузера
  • Privacy Policy ссылка видна без скролла
  • Имя бизнеса и контакт на странице
  • Загрузка <3 секунды на мобиле (проверить PageSpeed Insights)
  • Нет сексуализированных изображений
  • Нет слов: webcam, adult, OnlyFans, модель взрослого контента
  • Контент совпадает с обещанием объявления
  • CTA ведёт на WhatsApp / Bot / Форму (не на внешние платформы)

Объявление

  • Нет допущений о личных атрибутах читателя
  • Нет гарантий дохода ("зарабатывай $X гарантировано")
  • Нет прямого языка о найме (если не задана Employment Category)
  • Изображения: нейтральный контекст (не спальня, не сексуализировано)
  • Видео: 9:16, субтитры добавлены, хук в первые 3 секунды
  • CTA: "Saiba mais" / "Fale conosco" / "Enviar mensagem"
  • Employment Category не нужна (цель BR, нет явного job language)

Запуск

  • Запуск после 1–2 недель прогрева бренд-контентом
  • Начальный бюджет: $10–15/день (не весь $100 сразу)
  • Таргетинг: Женщины, 18–35, Бразилия, интересы: criadores, entretenimento, redes sociais
  • Account Quality dashboard открыт — проверять первые 3 дня ежедневно
  • Click-to-WhatsApp как основной формат
