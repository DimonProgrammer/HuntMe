📡 Конверсионный трекинг

Что установить ДО запуска рекламы

🔴 КРИТИЧНО: без трекинга реклама бессмысленна
Meta Pixel + события — обязательное условие. Без них нет данных для оптимизации, нет ретаргетинга, нет Lookalike.

Шаг 1: Meta Pixel на лендинге

  1. Создать Meta Pixel в Events Manager (business.facebook.com → Events Manager → Data Sources)
  2. Добавить base code в <head> лендинга (landing/index.html): 
    <!-- Meta Pixel Code -->
<script>
!function(f,b,e,v,n,t,s){...}(window,document,'script',
'https://connect.facebook.net/en_US/fbevents.js');
fbq('init', 'YOUR_PIXEL_ID');
fbq('track', 'PageView');
</script>
  3. Проверить через Meta Pixel Helper (Chrome extension)

Шаг 2: Событие Lead (клик на WhatsApp)

На кнопку «Fale com a Gabi» (WhatsApp link) повесить: 
onclick="fbq('track', 'Lead', {content_name: 'whatsapp_click'});"
Это ключевое событие. По нему Meta будет оптимизировать показ рекламы (когда переключимся на Lead objective).

Шаг 3: UTM-метки в рекламных ссылках

https://apextalent.pro/?utm_source=meta&utm_medium=paid&utm_campaign={{campaign.name}}&utm_content={{ad.name}}
Лендинг передаёт UTM в WA-ссылку → бот сохраняет в CRM → видим какой креатив привёл лида.

Шаг 4 (опционально, Фаза 2): CAPI

Conversions API (серверный трекинг) — подключить когда будет объём 50+ лидов/нед. Без CAPI теряем 30-40% данных из-за iOS ATT + блокировщиков. Реализация: бот при квалификации отправляет событие CompleteRegistration через CAPI.

Месячный план ($400/мес = ~R$2,200)

МесяцБюджетФокусОжидаемые лиды

Месяц 2 $400 Оптимизация: убить проигравших, масштабировать победителей. Запуск ретаргетинга 120-200
Месяц 3 $400 Масштабирование: Lookalike + лучшие креативы. Переход на Lead objective 150-250

Недельное распределение ($100/нед)

Фаза 1 (месяц 1):
$70 — Campaign 1 (Traffic, проспектинг)
$20 — Тест новых креативов
$10 — Резерв
Фаза 2+ (месяц 2-3):
$60 — Winning ad sets
$25 — Ретаргетинг / Lookalike
$15 — Тест новых хуков

Правила управления бюджетом

  • 3× Kill Rule: Если ad set потратил 3× целевого CPL и 0 лидов → пауза немедленно
  • 20% Scale Rule: Если CPL < таргета на 10%+ → поднять бюджет на 20% (не больше!)
  • Frequency > 4.0: Креатив выгорел → заменить или расширить аудиторию
  • Learning phase: Не трогать ad set первые 3-5 дней после изменений

🚀 Roadmap внедрения

Фаза 0 — Подготовка (3-5 дней)

До запуска рекламы

  • ☐ Создать Meta Business Suite аккаунт (личный, без юрлица)
  • ☐ Создать рекламный аккаунт в Meta Ads Manager
  • ☐ Создать Meta Pixel → установить на лендинг
  • ☐ Настроить событие Lead на WA-кнопку
  • ☐ Проверить Pixel через Meta Pixel Helper
  • ☐ Адаптировать лендинг под PT-BR (уже есть EN-версия)
  • ☐ Подготовить UTM-шаблоны для ссылок
  • ☐ Создать 4-6 креативов (Reels-видео + статика)
  • ☐ Привязать карту/PayPal для оплаты рекламы
Фаза 1 — Тестирование (Недели 1-3)

Запуск и поиск winning creative

  • ☐ Запустить Campaign 1 (Traffic) с 2 ad sets
  • ☐ По 3-4 разных креатива в каждом ad set
  • ☐ Бюджет: $10/день CBO
  • ☐ Мониторинг ежедневно первые 7 дней
  • ☐ День 4-5: выключить худшие креативы (CTR < 0.5%)
  • ☐ День 7-10: определить winning ad set и winning creative
  • ☐ День 14: 3× Kill Rule — убить всё что не конвертирует
  • ☐ Параллельно: проверить WA-бот на реальных BR-лидах
Фаза 2 — Оптимизация (Недели 4-6)

Масштабировать победителей

  • ☐ Увеличить бюджет winning ad set на 20%
  • ☐ Запустить ретаргетинг (посетители лендинга → WA)
  • ☐ Создать Custom Audience из WA-лидов (если 100+ контактов)
  • ☐ Запустить Lookalike 1% от лидов
  • ☐ Тестировать новые хуки (2-3 новых видео/неделю)
  • ☐ Анализировать: какой креатив → лучшие лиды (качество, show-up rate)
  • ☐ Рассмотреть переход objective Traffic → Leads (если 50+ конверсий/неделю)
Фаза 3 — Масштабирование (Недели 7-12)

Расширение и автоматизация

  • ☐ Подключить CAPI (серверный трекинг) для точной оптимизации
  • ☐ Расширить гео: P1 → P2 штаты
  • ☐ Тестировать Advantage+ (Meta AI полностью управляет таргетингом)
  • ☐ A/B тест лендингов (разные hero-блоки)
  • ☐ Запросить увеличение бюджета если unit economics положительная
  • ☐ Начать подготовку к CO/AR рынкам (испанский контент)
