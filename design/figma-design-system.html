13 — Do / Don't
✓
DO
· Use Outfit for ALL headings, numbers, labels
· Use .text-gradient for 1–2 key words per section max
· Apply card + hover lift to all surface containers
· Use rgba(26,15,46, N) for all tints — never grey hex
· Keep backgrounds always warm (cream gradient)
✗
DON'T
· Don't use Inter or system fonts — always load Outfit
· Don't use pure #000 or #FFF for text — use ink (#1A0F2E)
· Don't invent new accents outside ink/magenta/lime/orange
· Don't use border-radius less than 8px on interactive elements
· Don't add animation durations over 800ms for UI interactions
