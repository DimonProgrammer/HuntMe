## Техническое

- **Прогресс:** лид пишет `+` для продолжения (micro-commitment, сильнее кнопки — сам набирает)
- **Ветвления:** нумерация (`1`, `2`)
- **Произвольный текст:** AI (Groq + knowledge base) отвечает от лица Габи → продолжение воронки
- **Медиа:** фото, видео/GIF, аудио-заметки (PTT)
- **Паузы:** 2-3 сек между сообщениями (имитация печати)
- **Follow-up при молчании:** 1ч → 24ч → 72ч

### Системный промпт AI
```
Ты — Габи, 26 лет, бразильянка из Сан-Паулу. Ты ведёшь прямые эфиры

Тон: тёплый, энергичный, как подруга. Короткие предложения. Не более 3-4 строк.
Много эмодзи. Обращение на «ты».

Текущий шаг воронки: {шаг}
Контекст из базы знаний: {релевантный фрагмент из knowledge/}

Если вопрос = возражение, используй фреймворк A-A-R:
- Acknowledge (признай чувство)
- Answer (ответь по существу)

После ответа — CTA:
- «Отправь + и продолжим 💬»
```

| Шаги 24+ (квалификация) | «Мы почти закончили! Осталось {N} вопросов 😊» |

### Через 72 часа (финальное)

```
Привет! Просто хотела сказать — предложение никуда не денется.
Когда будешь готова, напиши. Я здесь 🙂
```

> После 72ч — больше не пишем. Лид остаётся в базе.

### Re-entry (лид вернулась)

| Пауза | Drop-off | Сообщение |
| --- | --- | --- |
| < 72ч | Шаги 0-6 | Рестарт с шага 0 |

### Graceful exit (явный отказ)

```
Понимаю! Это точно не для всех, и я уважаю решение

Если когда-нибудь захочешь вернуться — просто напиши 💛
```

> Пометить `declined`, исключить из follow-up.
