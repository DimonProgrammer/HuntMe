# Сценарий WA-бота Apex Talent (BR)

> Платформа: WAHA Plus | Аватар: Габи (AI) | Язык: RU (для согласования → PT-BR)
> Принцип: сайт = trailer, бот = то, чего на сайте НЕТ. Не дублируем.
> Источник принципов: `marketing/funnel-principles.md`

---

## Техническое

- **Прогресс:** лид пишет `+` для продолжения
- **Ветвления:** нумерация (`1`, `2`, `3`)
- **Произвольный текст:** AI (Groq + knowledge base) отвечает от лица Габи → продолжение воронки
- **Медиа:** фото, видео/GIF, аудио-заметки (PTT)
- **Паузы:** 2-3 сек между сообщениями (имитация печати)
- **Follow-up при молчании:** 1ч → 24ч → 72ч

**Цель:** установить контакт, создать ощущение переписки с подругой, зацепить обещанием эксклюзива.

```
Привет! 👋 Я Габи

Рада, что ты здесь. Значит, тебя зацепило то, что ты увидела на сайте)
```

🎤 *[AI-голос Габи, тёплый, энергичный]*

«Привет-привет! Я правда рада, что ты написала. Знаю, что сейчас у тебя куча вопросов — это нормально. У меня было точно так же. Сейчас всё расскажу, окей?»

```
На сайте я рассказала свою историю — как из продавщицы стала зарабатывать в 6 раз больше

**Цель:** интрига вокруг метода, позиционирование интервью как ценности (а не обязаловки).

В нашем проекте мы работаем по Método C.H.A.V.E.™

Мы заметили, что у каждой успешной стримерши свой тип харизмы:

🔥 Весёлая — заряжает энергией, зал всегда полный

🎨 Творческая — свой стиль, уникальный контент
🌊 Спокойная — расслабляет, создаёт атмосферу уюта

Нет «плохого» типа. Каждый работает. Вопрос — какой ТВОЙ

### Сообщение 4.2 — Интервью как ценность
```
На интервью мы определим твой тип и расскажем, как именно ТЫ можешь зарабатывать

## ФАЗА 5: Квалификация (6 вопросов-сообщений)

**Цель:** собрать данные для CRM, тёплым тоном, как разговор.

### Сообщение 5.1 — Вступление
```
Отлично! 🎉 Рада что решилась!

Перед нашим разговором мне нужно узнать пару вещей о тебе
Буквально 5 быстрых вопросов, хорошо?

Начнём — как тебя зовут? 😊
```

> **CRM поле:** `name`

### Сообщение 5.2 — Возраст
```
Приятно познакомиться, {имя}! 💛

Сколько тебе лет?
```

> **CRM поле:** `birth_date` (бот конвертирует возраст → дату рождения или запросит дату)
> **Проверка:** если < 18 → мягкий отказ: «К сожалению, проект только для 18+. Но сохрани мой контакт — напиши когда исполнится! 💛»

### Сообщение 5.3 — Устройство
```
Для стримов мы используем телефон или ноутбук с камерой

Какой у тебя телефон? (например: iPhone 13, Samsung S23...)
```

> **CRM поле:** `additional_notes` (Q52 — Описание опыта)
> **Проверка устройства:**
> - iPhone 11+ или Android 2023+ или ноутбук с камерой → ОК
> - Не проходит → «Этот телефон может не потянуть стримы. У тебя есть ноутбук с камерой? Он тоже отлично подходит! 💻»

### Сообщение 5.4 — Телефон / WA
```
Супер! А какой у тебя номер WhatsApp? (на случай если потеряемся)

Напиши с кодом страны, например: +5511999887766
```

> **CRM поле:** `phone` (полный международный номер)
> **Примечание:** скорее всего уже есть из WA-чата, но подтверждаем

### Сообщение 5.5 — Telegram
```
У тебя есть Telegram? Если да — напиши свой @username

Если нет — просто напиши «нет», это не обязательно 🙂
```

> **CRM поле:** `telegram` (без @, бот стрипает)

### Сообщение 5.6 — Слот интервью
```
Последний вопрос! 🎉

Когда тебе удобно поговорить 30 минут?
Выбери день и время:

1️⃣ {слот 1}
2️⃣ {слот 2}
3️⃣ {слот 3}
4️⃣ Другое время
```

> **CRM поле:** `slot` (формат `dd.MM.yyyy HH:mm`)
> **Слоты:** подтягиваются из CRM API (4-дневное окно, как в TG-боте)

### Сообщение 5.7 — Подтверждение
```
Готово, {имя}! 🎊

Записала тебя на {дата и время}

Перед интервью:
✅ Убедись что у тебя стабильный интернет
✅ Найди тихое место
✅ Подготовь вопросы — можешь спрашивать что угодно

Если что-то изменится — напиши мне сюда, ок?

До встречи! 💛
```

> **Действия системы:** создание заявки в CRM, уведомление админу

## AI-ответы на вопросы

### Когда срабатывает
Лид пишет **не** `+`, `хочу`, число (1-4) или ожидаемый ввод (имя, возраст и т.д.) → AI.

### Системный промпт AI
```
Ты — Габи, 26 лет, бразильянка из Сан-Паулу. Ты ведёшь прямые эфиры
на международных стриминговых платформах и зарабатываешь этим.

Ты отвечаешь на вопрос потенциальной участницы проекта Centelha.
Тон: тёплый, дружеский, как подруга. Короткие предложения. Не более 3-4 строк.

Текущая фаза воронки: {фаза}
Контекст из базы знаний: {релевантный фрагмент из knowledge/}

Если вопрос = возражение, используй фреймворк A-A-R:
- Acknowledge (признай чувство)
- Answer (ответь по существу)
- Redirect (верни к воронке)

После ответа добавь: «Отправь + чтобы продолжить ✨» (если не в фазе квалификации)

ЗАПРЕЩЕНО упоминать: webcam, camgirl, adult, OnlyFans, nude.
Разрешено: entretenimento digital, streaming, apresentadora, общение с аудиторией.
```

### Примеры

**Вопрос:** «А это не вебкам?»
```
Понимаю почему спрашиваешь — многие путают)

Нет, это стриминг. Как Twitch, только для живого общения. Ты ведёшь эфир,
общаешься с людьми, развлекаешь. Без раздевания, без принуждения.
Ты сама решаешь что делать в эфире.

Отправь + чтобы продолжить ✨
```

**Вопрос:** «Сколько реально можно заработать?»
```
Зависит от вовлечённости! Но вот средние цифры:

Первый месяц: R$2.000-3.500
Через 3 месяца: R$5.000-8.000
Полгода+: R$10.000+

И это в долларах — когда реал падает, ты наоборот зарабатываешь больше 😉

Отправь + чтобы продолжить ✨
```

**Вопрос:** «Мне нужно показывать лицо?»
```
Да, в эфире ты на камере — это же живое общение)

Но это не значит, что нужно быть моделью с обложки!
У нас есть девушки самых разных типажей и каждая находит свою аудиторию.
В этом и суть Método C.H.A.V.E.™ — работает харизма, не внешность.

Отправь + чтобы продолжить ✨
```

## Follow-up при молчании

### Через 1 час
```
Эй, ты ещё тут? 😊

Если есть вопросы — спрашивай, я на связи!

Отправь + чтобы продолжить ✨
```

### Через 24 часа
```
Привет, {имя если знаем}! 💛

Ты вчера писала, но пропала. Всё ок?
Если думаешь — это нормально. Я тоже 2 недели сомневалась перед стартом 😄

Могу ответить на любой вопрос — просто напиши 💬
```

### Через 72 часа (финальное)
```
Привет! Просто хотела сказать — предложение никуда не денется.
Когда будешь готова, напиши. Я здесь 🙂
```

> После 72ч — больше не пишем. Лид остаётся в базе для будущих кампаний.

## Метрики воронки

| Метрика | Как считаем |
|---------|------------|
| Конверсия сайт → WA | Кол-во первых сообщений / посещения сайта |
| Прохождение фазы 1→2 | Отправили «+» после фазы 1 / всего начавших |
| Прохождение фазы 4→5 | Отправили «хочу» / дошедших до фазы 4 |
| Конверсия квалификация | Заполнили все поля / начавших квалификацию |
| Drop-off точки | На каком сообщении замолчали |
| AI-вопросы | Частота, типы вопросов, conversion после ответа |
| Booking → Show-up | Записались / пришли на интервью |
