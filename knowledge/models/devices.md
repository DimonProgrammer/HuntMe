# Требования к устройствам для моделей

> Источник: docs/model-application-guide.md, созвон 2 марта

---

## Общий принцип

Для работы стриминговой модели нужно **одно из двух**:
1. Смартфон с хорошей камерой (основной вариант)
2. Ноутбук/ПК с веб-камерой (альтернативный вариант)

Если у кандидата есть **и** смартфон, **и** ноутбук с камерой — заявка одобряется с максимальным приоритетом.

---

## iPhone — требования

| Критерий | Значение |
|---|---|
| Минимальная модель | **iPhone 11** |
| Подходят | iPhone 11, 12, 13, 14, 15, 16 (все модели серии: mini, Pro, Pro Max и т.д.) |
| НЕ подходят | iPhone X, XR, XS, 8, 7, SE и старше |

### Важные нюансы

- **"Просто iPhone" без указания модели = ОТКАЗ.** Необходимо знать точную модель. "У меня iPhone" — недостаточно
- Точная модель критически важна, т.к. старые iPhone не поддерживают нужное качество видео
- При заполнении CRM-заявки обязательно указывать конкретную модель (iPhone 13 Pro, iPhone 15 и т.д.)

---

## Android — требования

| Критерий | Значение |
|---|---|
| Минимальный год выпуска | **2023** |
| Ключевой фактор | Год выпуска устройства, НЕ бренд |

### Подходящие Android-устройства (примеры)

- Samsung Galaxy S23/S24/S25, Galaxy A54/A55 и новее
- Xiaomi 13/14/15, Redmi Note 12 Pro+ и новее (2023+)
- Google Pixel 7a/8/9 и новее
- OnePlus 11/12/13 и новее
- Любой Android-смартфон 2023 года выпуска и новее с качественной фронтальной камерой

### НЕ подходят

- Любые Android-устройства до 2023 года выпуска
- Бюджетные модели с плохой фронтальной камерой (даже если 2023+) — на усмотрение интервьюера

---

## Ноутбук/ПК + веб-камера

Альтернативный вариант для работы. Если у кандидата нет подходящего смартфона, но есть ноутбук/ПК с веб-камерой — **это 100% одобрение** по устройству.

| Требование | Описание |
|---|---|
| Веб-камера | Обязательно. Встроенная в ноутбук или внешняя USB |
| Качество камеры | HD (720p) минимум, предпочтительно Full HD (1080p) |
| ОС | Без ограничений (Windows, macOS, Linux) — для моделей MacBook допустим |

**Важно:** Для моделей MacBook подходит (в отличие от операторов, где MacBook = отказ). Модели НЕ работают с OBS и специализированным стриминговым ПО.

---

## Дополнительное оборудование

| Оборудование | Обязательно? | Комментарий |
|---|---|---|
| Наушники с микрофоном | Желательно | Не блокер. Любые наушники с микрофоном подойдут, стоят до $10 |
| Стабильный интернет | Да | Для стабильного видеопотока |
| Отдельная комната | Да | Hard requirement — без посторонних в кадре |

---

## Интернет

- Стабильное соединение — обязательно
- Конкретный порог скорости для моделей не установлен (в отличие от операторов, где 100 Mbps+)
- Если скорость низкая, но кандидат готов улучшить — **ОК** (единственное исключение из hard requirements)

---

## Логика одобрения устройств

```
Подходящий телефон (iPhone >= 11, Android 2023+)
    → ОДОБРЕНИЕ

Ноутбук/ПК с веб-камерой
    → ОДОБРЕНИЕ (100%)

Подходящий телефон + ноутбук с камерой
    → ОДОБРЕНИЕ (максимальный приоритет)

Только "iPhone" без указания модели
    → ОТКАЗ (уточнить модель → если >= 11, пересмотреть)

Только "Android" без указания модели/года
    → ОТКАЗ (уточнить модель → если 2023+, пересмотреть)

Старый телефон + нет ноутбука
    → ОТКАЗ

Обещание купить устройство после собеседования
    → ОТКАЗ

Низкая скорость интернета, готов улучшить
    → ОК (единственное исключение)
```

## Частые ситуации

| Ситуация | Решение |
|---|---|
| "У меня iPhone" (без модели) | Уточнить: "Какая именно модель? iPhone 11, 12, 13...?" |
| "У меня Samsung" (без модели) | Уточнить: "Какая модель? Samsung Galaxy S...? Когда покупали?" |
| "Куплю телефон после собеса" | Отказ. Устройство должно быть на руках |
| "У меня iPad / планшет" | Уточнить: есть ли смартфон или ноутбук. Планшет — на усмотрение |
| "Телефон старый, но есть ноутбук с камерой" | Одобрение по ноутбуку |
| "iPhone 11, плюс есть MacBook" | Одобрение с максимальным приоритетом |
