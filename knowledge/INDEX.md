## Правила

1. Каждый файл < 500 строк — AI может загрузить целиком
2. Заголовки = навигация — AI ищет по заголовкам, потом углубляется
3. Формат: только MD (HTML конвертирован)
4. Язык: EN для скриптов/контента, RU для внутренних гайдов
5. Каждый файл содержит «Источник:» — откуда взята информация
6. Approve/Reject кандидатов — ТОЛЬКО ручной (AI не авто-аппрувит)
